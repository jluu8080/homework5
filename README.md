# Homework 5

Homework 5 has 3 categories of requirements:
1. Sass Requirements
2. Layout Requirements
3. Responsive/Responsiveness Requirements  


1.Sass Requirements (At most):
- One variable file
- One mixins file
- One additional partial file
- Nested Selectors (where needed/appropriate)
- At least one custon mixin  

2.Layout Requirements:
(Use Both)
-  CSS Grid
- Flexbox

3.Resposnive/Responsiveness Requirements:
You must include resposive layouts for the following devices:
- Desktop
- Tablet
- Mobile Devices:

Mobile Responsive Layout must do the follow:
- Display a navigation icon insteagd of the full navigation menu.

- Stack contact vertically (up & down) in a single column

- Maintain readable typography & spacing.
